Sublime Text 설정

키 바인딩
Ctrl + `를 눌러 콘솔 창을 열고 다음 명령어를 입력하고 엔터 키를 누른다.

이것은 서브라임 텍스트에서 일어나는 모든 행동을 기록하라는 명령어 이다. 참고로 주의할 부분은 True는 대문자로 입력해야 한다.

명령어를 알아내면 콘솔 기록을 중지하기 위해 다음과 같은 명령어를 입력하고 엔터 키를 누른 다음 Esc 키를 눌러 콘솔 창을 닫는다.

사용자 키맵을 열고 마지막 줄에 콤마를 입력한 다음 아래 코드를 입력하고 저장하면 메뉴에 단축키가 나타나게 된.

커맨드 라인에서 서브라임 텍스트 실행하기
시스템설정 -> 고급 탭 -> 환경변수 Path에 서브라임텍스트 (설치된)경로를 등록한다.

관리자 권한으로 cmd 실행 후

경로가 바뀌면 다시 아래 명령을 입력하고 엔터키를 누른다.

그러면 기호화된 링크(Symbolic Link)가 만들어 졌다는 메세지가 나타나게 된다.
그럼 이제 subl만 입력하고 엔터 키르 누르면 서브라임 텍스트가 실행된다.

Git을 사용할 경우 Git Bash에서도 작동한다.
