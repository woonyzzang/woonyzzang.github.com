IIS7 에서 SSI(server side include) 사용하기

SSI란? Server Side Include의 약어로, 웹서버에서 직접 제공하는 서버사이드 스크립트 언어다.

IIS 7 기본 셋팅에서는 include 를 해도 인클루드 기능이 적용이 되지 않는다.

IIS7 에서 SSI를 적용하려면 다음과 같다.

  1. IIS 역할 서비스 추가 > 응용 프로그램 개발 > SSI(Server Side Include)의 기능을 설치해야 한다.

  1. 설치가 완료되면 IIS 관리자 > Handler Mappings 메뉴를 선택한다.

  1. 우측 Actions 패널의 Add Module Mapping... 메뉴를 선택한다.

  1. IIS 처리기 맵핑에 해당 모듈을 추가한다.
